# IronLink — Mobile UI/UX Design Mockups

Static HTML mockups of the IronLink mobile app (a jobs / pay-transparency /
community / equipment-marketplace app for US skilled trades), built on a single
shared design system.

**Live site:** https://vitech-team.github.io/ironlink-design/

- `index.html` — screen gallery (all screens grouped by journey)
- `flows.html` — user-flow walkthroughs (screens connected into journeys)
- `screens/` — 55 individual screen mockups
- `styles/ironlink.css` — the design system (single source of truth)
- `_spec/` — written specs (offline model, dialogs, negative cases, motion & IA, nav decisions)

Open `index.html` locally, or browse the live site above.
